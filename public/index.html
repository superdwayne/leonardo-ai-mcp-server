🎨 Leonardo AI MCP Server Remote

A remote Model Context Protocol (MCP) server for Leonardo.ai, powered by Vercel.

Available Tools

  • generate_image — Generate images using Leonardo AI models
  • get_generation — Get the status and results of a generation job
  • list_models — List available Leonardo AI platform models
  • get_user_generations — Get recent generation jobs for the current user
  • get_user_info — Get the current authenticated user's information
  • create_variation — Create a variation of an existing generated image
  • upscale_image — Upscale a generated image
  • delete_generation — Delete a generation by ID

Connect via MCP

Point your MCP client to:

https://your-deployment-url.vercel.app/mcp

Cursor / Claude Desktop Configuration

{
  "mcpServers": {
    "leonardo-ai": {
      "url": "https://your-deployment-url.vercel.app/mcp",
      "headers": {
        "x-leonardo-api-key": "YOUR_LEONARDO_API_KEY"
      }
    }
  }
}
